В электронных кругах можно делать все - обсуждать вопросы, обмениваться документами, распределять задачи внутри команды, контактировать с клиентами.

На собственном сервере вы можете делать круги с помощью разных инструментов.

Сетевая папка

Назначайте коллегам права для чтения и записи к сетевой файловой папке для совместной работы над документами в простом и удобном интерфейсе. С помощью программы QNAP QSync каждому участнику доступна синхронизация сетевой папки с папкой на своем ноутбуке, планшете или мобильном телефоне. На сервере QNAP ведутся журналы доступа к сетевым папкам чтобы видеть кто и что изменил.

Доступ к сетевой папке возможен по всем популярным протоколам для компьютеров на MacOS, Windows и для мобильных телефонов на iOS и Android. Конечно, доступ возможен и через любой веб-браузер.

Чтобы информация в папке была читабельной, можно ввести некоторые условности. Например, внутри заводить папки в символом подчеркивания для обозначения значимости. "_11.11.2020" - название для временной вехи начала или срока задачи, "_Отчет по спринту" - для обозначения результата, "_Фамилия Имя Отчество (клиент)" - для ключевых контактов. Даже пустая папка с подчеркиванием в названии может сразу давать информацию о контексте работы команды, но в такую папку можно разместить важные копии писем или результирующих документов.

При использовании QSync видны недавно измененнные документы с возможностью быстрого перехода к ним. Также можно настроить версионное хранение для дубликатов и ничего не терять.

Команда в чате Mattermost

Идеальной заменой почты для интенсивной коммуникации служит бизнес-чат с быстро настраиваемыми командами, общими и приватными каналами с доступом для выбранных контактов.

Как и в почте, сообщение можно наполнить приложенными документами, указать адресат в определенном месте текста сообщения. В отличие от почты не нужно настраивать фильтры для перемещения сообщений по почтовым папкам. Теперь папки общие для выбранного круга контактов, и все получают письма сразу в нужную папку с удобным уведомлением в мобильном приложении или в веб-браузере.

Удобно пригласить клиента в выделенный для него канал и подключить нужных специалистов. Приглашение можно выслать в виде ссылки любым удобным способом - по почте или мобильному мессенджеру. При этом, вся переписка и история переписки ведется прямо на вашем сервере QNAP.

Проект Redmine

Некоторые задачи требуют координации действий команды во времени. При этом важно понимать, как движется команда по определенным метрикам. Например, как идет процесс набора новых сотрудников, или как движется проект разработки сайта по определенным этапам и запланированным часам привлеченных специалистов.

В программе Redmine электронный круг формируется с помощью проекта. Приглашения рассылаются по электронной почте. А работа с программой возможна с помощью веб-браузера или через мобильное приложение.

В Redmine есть мощная поддержка работы как с проектами, так и с процессами. А интерфейс привычно напоминает электронные таблицы.

Проект Gitlab

Программа GitLab позволяет гибко разрабатывать любую документацию. Придуманная для программистов, программа идеально подходит для версионной коллективной разработки документов с текстовой разметкой.

Электронный круг в GitLab формируется в виде проекта с поддержкой задач, участников проекта с определенными полномочиями, фильтрацией по тегам и вехам.

Встроенная поддержка версионной системы Git позволяет создавать ветвления в работе над документами, проводить контролируемое слияние ветвей и видеть зоны работы команды над документами.
